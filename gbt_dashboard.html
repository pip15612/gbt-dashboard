📖 คู่มือ GBT Analysis

อ่านค่า Dashboard ตั้งแต่ระดับพื้นฐานจนถึง Financial Engineering · ทุกสูตรใช้ Black-76 (r=0, Options on Futures)

🎯 ส่วนที่ 1 — พื้นฐาน สำหรับผู้เริ่มต้น

📍 ATM (At-The-Money) คืออะไร?

ATM คือ ราคา Strike ที่ใกล้เคียงกับราคา Futures ปัจจุบันมากที่สุด เช่น ถ้า Gold Futures อยู่ที่ 4,540 — Strike 4,540 คือ ATM

ใน dashboard นี้ Futures Price = ราคา ATM ที่ดึงจากหัว CSV โดยตรง (บรรทัด "vs 4540.5")

💡 ATM คือจุดอ้างอิงหลัก — ราคาทุกอย่างวัดจากจุดนี้
📍 DTE (Days to Expiry) คืออะไร?

DTE = จำนวนวันก่อนที่ Option จะหมดอายุ เช่น DTE 0.46 = เหลือเวลาไม่ถึงครึ่งวัน

ผลต่อ Options:

• DTE สูง → Premium มาก → Gamma ต่ำ (ราคาเปลี่ยนช้า)
• DTE ต่ำ → Premium น้อย → Gamma สูงมาก (ราคาเปลี่ยนเร็ว เสี่ยงสูง)

⚠️ เมื่อ DTE < 1 วัน Gamma จะ spike สูงมาก — ราคา Options ผันผวนรุนแรง
📍 IV — Implied Volatility (Vol Settle) คืออะไร?

IV คือ ความผันผวนที่ตลาดคาดการณ์ไว้ในราคา Option — ไม่ใช่ความผันผวนในอดีต แต่เป็นการมองไปข้างหน้า

ตัวเลขใน column Vol Settle ใน CSV เป็น decimal เช่น 0.1828 = 18.28%

ตัวเลขใน header2 "Vol: 17.83" เป็น เปอร์เซ็นต์ โดยตรง

1σ Expected Range (Expiry) = F × σ × √(DTE/252)

ตัวอย่าง: F=4540, σ=18.28%, DTE=0.46 → Range = 4540 × 0.1828 × √(0.46/252) ≈ ±$25

💡 IV สูง = ตลาดกังวล ค่า premium แพง / IV ต่ำ = ตลาดสงบ ค่า premium ถูก
📍 Call / Put Volume และ Open Interest คืออะไร?

Call Volume = จำนวน Contract ที่ซื้อขาย Call Option วันนี้ (Intraday)
Put Volume = จำนวน Contract ที่ซื้อขาย Put Option วันนี้ (Intraday)

Call OI = จำนวน Call Contract ที่ยัง open อยู่ทั้งหมด
Put OI = จำนวน Put Contract ที่ยัง open อยู่ทั้งหมด

💡 Volume บอกว่าวันนี้ซื้อขายมากแค่ไหน / OI บอกว่ามีคนถือค้างไว้เท่าไหร่

⚡ ส่วนที่ 2 — GEX (Gamma Exposure) หัวใจของ Dashboard

🔬 GEX คืออะไร? และมาจากไหน?

GEX (Gamma Exposure) คือ การวัดแรงกดดันของ Dealer (Market Maker) ต่อราคา ที่แต่ละ Strike

Dealer ขาย Option ให้นักลงทุน → Dealer ต้อง hedge ด้วย Futures → Gamma บอกว่าต้อง hedge เท่าไหร่เมื่อราคาเปลี่ยน

Gamma (Black-76) = N'(d₁) / (F × σ × √T)

d₁ = [ln(F/K) + ½σ²T] / (σ√T)

Net GEX (per strike) = Gamma × (Call OI − Put OI) × F² × 100

F = Futures price, K = Strike, T = DTE/365, σ = IV (decimal), N'(x) = Standard Normal PDF, 100 = oz per GC contract

💡 คิดง่ายๆ: GEX บอกว่า Dealer จะ "เหยียบเบรก" หรือ "เหยียบคันเร่ง" ตลาดที่แต่ละ Strike
🟢 GEX บวก vs 🔴 GEX ลบ — หมายความว่าอะไร?

GEX บวก (Call dominant) — ที่ Strike นั้น Call OI > Put OI
→ Dealer Short Call → เมื่อราคาขึ้น Dealer ต้อง Sell Futures (เบรกการขึ้น)
→ เมื่อราคาลง Dealer ต้อง Buy Futures (เบรกการลง)
ผล: ราคาถูกดูดกลับ → ตลาด Mean-Revert ในโซน +GEX

GEX ลบ (Put dominant) — ที่ Strike นั้น Put OI > Call OI
→ Dealer Short Put → เมื่อราคาลง Dealer ต้อง Sell Futures ด้วย (ยิ่งลง)
→ เมื่อราคาขึ้น Dealer ต้อง Buy Futures ด้วย (ยิ่งขึ้น)
ผล: ราคาถูกดัน → ตลาด Trend ต่อในโซน -GEX

💡 สรุป: โซน +GEX = ดูดราคา / โซน -GEX = ดันราคา
🎯 Call Wall, Put Wall, DGC Wall คืออะไร?

Call Wall (+Wall) = Strike ที่มี GEX บวกสูงสุด → Dealer มีแรงกดดัน Sell Futures มากที่สุด → แนวต้านที่แข็งแกร่ง

Put Wall (−Wall) = Strike ที่มี GEX ลบมากที่สุด → Dealer มีแรงกดดัน Sell Futures มากเมื่อราคาลง → แนวรับที่แข็งแกร่ง

DGC Wall = จุดกึ่งกลางระหว่าง Call Wall และ Put Wall

DGC Wall = (Call Wall Strike + Put Wall Strike) / 2
💡 DGC Wall = "จุดสมดุล" ของตลาด — ราคามักวนรอบจุดนี้ในวัน
⚡ Gamma Flip คืออะไร? ทำไมสำคัญมาก?

Gamma Flip = จุดที่ Cumulative GEX เปลี่ยนเครื่องหมาย จาก + เป็น − (หรือกลับกัน)

ความสำคัญ:

• ราคา อยู่เหนือ Gamma Flip → Net GEX บวก → Dealer Long Gamma → ตลาด Mean-Revert ความผันผวนต่ำ

• ราคา อยู่ต่ำกว่า Gamma Flip → Net GEX ลบ → Dealer Short Gamma → ตลาด Trend ความผันผวนสูง

⚠️ เมื่อราคาหลุด Gamma Flip → มักเกิด momentum รุนแรง (เหมือน "ระเบิด Gamma Bomb")
💡 ใช้ Gamma Flip เป็น Key Level สำคัญ — อยู่เหนือ = ซื้อขายแบบ Range / ต่ำกว่า = ระวัง Trend

📐 ส่วนที่ 3 — Expected Range (ช่วงราคาที่คาดการณ์)

📏 1σ Range คืออะไร และใช้ยังไง?

1σ Expected Range = ช่วงราคาที่มีโอกาส 68.2% ที่ราคาจะอยู่ใน range นี้ก่อน Expiry

1σ Range (Expiry) = F × σ_ATM × √(DTE / 252)
1σ Range (1 Day) = F × σ_ATM × √(1 / 252)

ตัวอย่าง: F=4540, σ=18%, DTE=5
1σ Exp = 4540 × 0.18 × √(5/252) = 4540 × 0.18 × 0.1409 ≈ ±$115
ราคามีโอกาส 68% อยู่ระหว่าง 4,425–4,655 ก่อน Expiry

💡 ขาย Options ที่อยู่นอก 1σ มีโอกาสกำไรถึง 68% ตาม Theory (แต่ต้องดู GEX ประกอบด้วย)
📊 2σ Range คืออะไร?

2σ Range = ช่วงราคาที่มีโอกาส 95.4% ที่ราคาจะอยู่ใน range นี้

2σ Range = 2 × (F × σ_ATM × √(DTE / 252))
💡 ราคาที่หลุด 2σ ถือว่า "ผิดปกติมาก" — มักเกิดจาก Event พิเศษ (Fed, War, Panic)
⚡ Composite GEX และ α Slider คืออะไร?

GBT ใช้ข้อมูล 2 แหล่ง: Intraday Volume (ซื้อขายวันนี้) และ Open Interest (ถือค้างไว้)

ค่า α ช่วยผสมทั้งสอง:

Composite GEX = (1 − α) × GEX_OI + α × GEX_Volume

α = 0 → ใช้ OI 100% (structural, long-term)
α = 0.4 → ผสม 60% OI + 40% Vol (recommended)
α = 1 → ใช้ Volume 100% (intraday sentiment)
💡 ค่า 0.4 ดีที่สุดในส่วนใหญ่ — เน้น structure ระยะยาวแต่รับ signal ล่าสุดด้วย

📈 ส่วนที่ 4 — อ่านกราฟและสรุปสัญญาณ

🗺️ อ่านกราฟ GEX อย่างไร?

แนวตั้ง (ชนิดเส้น):
— ATM เส้นทึบสีทอง = ราคา Futures ปัจจุบัน
+Wall เส้นเขียว = Call Wall (แนวต้าน)
−Wall เส้นแดง = Put Wall (แนวรับ)
DGC Wall เส้นทองประ = จุดสมดุล

โซนสี (พื้นหลัง):
โซนม่วงเข้ม = 1σ Expiry Range (68% โอกาส)
โซนม่วงจาง = 2σ Expiry Range (95% โอกาส)

💡 สัญญาณซื้อ: ราคาอยู่เหนือ Gamma Flip + ใกล้ Put Wall + อยู่ใน 1σ Zone
💡 สัญญาณขาย: ราคาอยู่ใกล้ Call Wall + GEX บวกสูงมาก + ราคาใกล้ขอบ 1σ
∑ อ่านกราฟ Cumulative GEX อย่างไร?

กราฟ Cumulative GEX แสดงการรวม GEX สะสมจากซ้ายไปขวา (จาก Strike ต่ำไปสูง)

• เส้นอยู่เหนือ 0 = GEX รวมบวก → ราคาในโซนนั้นถูก "ดูด" กลับ
• เส้นอยู่ต่ำกว่า 0 = GEX รวมลบ → ราคาในโซนนั้นถูก "ดัน" ต่อ
• จุดที่เส้นข้ามศูนย์ = Gamma Flip → จุด key ที่สำคัญที่สุด

💡 ดูว่าราคาปัจจุบัน (ATM) อยู่ฝั่งไหนของ Gamma Flip — นั่นคือ "mode" ของตลาดวันนี้
🎯 สรุปกลยุทธ์การอ่านค่า GBT

ขั้นตอนการอ่านค่าทุกเช้า:

1. ดู ATM vs Gamma Flip — อยู่เหนือ Flip = Range mode / ต่ำกว่า = Trend mode

2. ดู 1σ Range — นี่คือ "สนามเด็กเล่น" ของราคาวันนี้

3. ดู Call Wall และ Put Wall — แนวต้าน/แนวรับหลัก

4. ดู DGC Wall — ถ้าราคาวนรอบนี้ = ตลาดไม่มี Direction

5. ดู Net GEX รวม — บวกมาก = ตลาดนิ่ง / ลบมาก = ระวัง

📌 GBT เป็นเครื่องมือ Flow Analysis — ใช้ประกอบกับ Price Action และ Macro Context เสมอ

🎯 ส่วนที่ 5 — Today's Edge (Zone Decision Engine)

🎯 Panel นี้ทำงานยังไง?

Today's Edge เป็น panel เด่นบนสุดของ tab GEX และ Cumulative ทำหน้าที่สรุป signal ทั้งหมดในจอเดียวเพื่อบอกว่า วันนี้ควรเทรดโซนไหน

ส่วนประกอบ (ซ้าย → ขวา):

• Verdict — LEAN/STRONG LONG (เขียว), LEAN/STRONG SHORT (แดง), หรือ NO TRADE / WAIT (เทา) พร้อม Confidence 0-100
• Trade Plan — Entry range, Stop, Target พร้อม R:R และจำนวน lot ตาม Risk ที่ตั้งไว้ที่ Zone Planner
• Signal Breakdown — 6 เหตุผลหลักที่ engine ใช้ตัดสิน (✓ บวก / ✗ ลบ / ⚠ เตือน) พร้อมคะแนน ±N
• Meta chips — GEX sign, Regime, Gap bucket, σ-D triggers, Vol-OI Quadrant, Daily EM, VRP

Confidence = max(scoreLong, scoreShort) · Verdict = Long ถ้า scoreLong − scoreShort > 8
ถ้า winnerScore < 18 หรือ |margin| < 8 → NO TRADE
💡 ภาพรวมการอ่าน: Verdict สีเขียว/แดงเข้ม + confidence > 50 = signal แรง / NO TRADE = อย่าฝืน รอราคาทะลุ ±1σ Trigger ก่อน

📊 Call/Put Change vs Previous Snapshot ใช้ยังไง?

การ์ดนี้แสดงผลต่างของ Call OI / Put OI ระหว่าง snapshot ปัจจุบันกับ snapshot ก่อนหน้า (ส่วนใหญ่ห่างประมาณ 30 นาที)

อ่านอย่างไร:

• ΔCall Total +ve = มีการเปิดสถานะ Call ใหม่เพิ่มเข้ามา (ตลาดเริ่มเก็งฝั่งขาขึ้น)
• ΔPut Total +ve = มีการเปิดสถานะ Put ใหม่ (กลัวลงหรือ hedge)
• ΔCall >> ΔPut → 📈 Call Build (Bullish flow)
• ΔPut >> ΔCall → 📉 Put Build (Bearish flow)
• ทั้งสองลง → ⬇ Unwind (ปิดสถานะก่อนเหตุการณ์ /ใกล้หมดอายุ)

🎰 ส่วนที่ 6 — Block Strategy ($25 / $50 / $100)

🎰 ทำไมต้องเล่นที่ Block $25 / $50 / $100?

จากข้อมูล Vol2Vol — Strike ที่ราคาลงท้ายด้วยตัวเลขกลม ($25, $50, $100) มี OI และ Volume กระจุกหนักอย่างมีนัยสำคัญ — เกิดจาก Round-Number Bias ของนักลงทุน (ทั้ง retail และ institutional ชอบเปิดสถานะที่ราคากลม)

สิ่งนี้สร้าง กำแพง Gamma ที่:

• ราคาเข้าใกล้ → Market Maker hedge หนักขึ้น → เด้งกลับชัด (mean revert)

• ราคาทะลุได้ → ทะลุแรง (Squeeze / Cascading) ไปหา Block ถัดไป

นี่คือเหตุผลที่ Median Daily Range ของทอง = $25.6 ตรงกับ $25 Strike Interval ของ CME แบบเป๊ะๆ

🔄 ส่วนที่ 7 — Recovery Trade (Flip + Lot Increase)

🔄 หลักการ Recovery: Flip ทาง + เพิ่ม Lot

เมื่อ SL ที่ ครึ่ง block ($12.5) — ราคามักเด้งกลับทางเดิมในวันเดียวกันตามสถิติ Intraday Reversion 54.7% (สูงถึง 64.4% ถ้า gap > $15) เราใช้กลไกนี้กลับมาเก็บ block แทน

ตัวอย่าง flow (SL=TP=$12.5):

1. Buy 0.10 lot ที่ $4,540 → SL $4,527.5 → ขาดทุน $125
2. Flip — Sell 0.20 lot (เพิ่มเป็น 2 เท่า) ที่ $4,527.5 → TP $4,515 → กำไร $250
3. Net = +$250 − $125 = +$125 (= 1 block profit)

ถ้าครั้งที่ 2 ก็โดน SL อีก:

• Cum loss = $125 + $250 = $375
• Trade #3 — Buy 0.40 lot → TP $12.5 = $500
• Net = $500 − $375 = +$125 ✓

Next Lot = (Cumulative Loss + Target Profit) ÷ (TP × $100)
ที่ Setup 1:1 → Lot จะโตเป็น 1, 2, 4, 8, 16… (เหมือน Martingale)
⚠ คำเตือนสำคัญ: Recovery 1:1 ใช้ Martingale ซึ่ง lot โตทวีคูณ — ถ้าพบ Gap หรือ Fat-tail event 1 ครั้งอาจล้างพอร์ตได้ ต้องตั้ง max sequence ≤ 5-6 ครั้ง แล้วยอม cut loss
💡 Setup 1:2 (SL $12.5 / TP $25) ปลอดภัยกว่ามาก เพราะ lot ไม่ต้องเพิ่มเร็ว (รอบแรก 0.10 → รอบ 2 = 0.10 พอ, รอบ 3 = 0.15)
🧮 ใช้ Recovery Lot Calculator (ใน Zone Planner)

ปรับ Initial Lot, SL, TP, Max sequence → ตารางจะคำนวณให้:

• Lot ที่ต้องใช้ในแต่ละ trade
• If TP = Net profit (รวม recover loss แล้ว)
• If SL = Cumulative Loss ที่ต้องแบกถ้าทุกไม้แพ้
• Margin = หลักประกันรวมโดยประมาณ (สมมติ $10,000/lot)

วิธีตั้ง Max sequence: ดู Worst-case Loss ใน summary — ถ้ามากกว่า 20-30% ของพอร์ต อย่าใช้ sequence ยาวกว่านั้น

💡 กฎทอง: ก่อนเริ่ม Recovery ในแต่ละวันให้ตั้งเป้า "จบที่ trade ที่เท่าไหร่ถ้ายังแพ้" เช่น 4 แล้วยอมรับ loss — ไม่ว่ายังไงก็ตาม
